# Wynnmaze
[![GitHub downloads](https://img.shields.io/github/downloads/FengLingYu7563/Wynnmaze/total?label=downloads&color=FFBB77&logo=github&logoColor=white)](https://github.com/FengLingYu7563/Wynnmaze/releases)[![Modrinth Downloads](https://img.shields.io/modrinth/dt/jBOBSrqW?color=FFBB77&label=modrinth&logo=modrinth)](https://modrinth.com/mod/wynnmaze)[![License](https://img.shields.io/github/license/FengLingYu7563/Wynnmaze?color=brightgreen&logo=github)](https://github.com/FengLingYu7563/Wynnmaze/blob/main/LICENSE)  
  
  
WynnMaze is a Wynncraft mod that enhances the player experience in the TCC raid's Room 2 maze by providing beginner-friendly gameplay for those new to the raid.

WynnMaze is based on Minecraft 1.21.11 and supports Fabric.

此模組是為 Wynncraft 的 TCC 副本的第二關的迷宮關設計的模組，讓新手更容易遊玩，支援 Fabric 1.21.11。
